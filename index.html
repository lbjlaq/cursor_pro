使用说明

系统要求

  • Google Chrome 浏览器
  • 管理员权限
  • Cursor Web端必须登录账号
  • 必须能正常访问 Cursor 官网
  • 稳定的网络连接

快速开始

  1. 下载并安装程序
  2. 以管理员身份运行
  3. 完成必要配置
  4. 开始使用功能

Cloudflare 邮箱配置

  1. 将域名托管到 Cloudflare
  2. 在 Email Routing 中启用邮箱服务
  3. 添加 Catch-all 规则
  4. 设置转发地址为 tempmail.plus 邮箱
  5. 等待 DNS 记录生效（约5-10分钟）

Tempmail.plus 配置

  1. 访问 tempmail.plus
  2. 选择一个自定义邮箱前缀
  3. 设置并记住 PIN 码
  4. 将邮箱地址和 PIN 填入助手配置
  5. 测试邮箱是否能正常接收邮件
⚠️ 注意：配置邮箱服务前，请确保：
  • DNS 记录已正确配置
  • 邮箱服务已正常启用
  • 已测试邮件转发功能
