# Cursor Pro Assistant

一款优雅、高效的 Cursor 辅助工具，支持 Windows 和 macOS (Apple Silicon)。

## ✨ 特性

- 🚀 **全自动注册**: 快速完成 Cursor 账号注册流程
- 🔄 **一键重置**: 轻松重置机器码，无需手动操作
- ⚙️ **可视化配置**: 简单直观的配置管理界面

## ⚙️ 配置说明

### Cloudflare 邮箱配置
1. 将域名托管到 Cloudflare
2. 在 Email Routing 中启用邮箱服务
3. 添加 Catch-all 规则
4. 设置转发地址为 tempmail.plus 邮箱
5. 等待 DNS 记录生效（约5-10分钟）

### Tempmail.plus 配置
1. 访问 [tempmail.plus](https://tempmail.plus)
2. 选择一个自定义邮箱前缀
3. 设置并记住 PIN 码
4. 将邮箱地址和 PIN 填入助手配置
5. 测试邮箱是否能正常接收邮件

⚠️ **注意事项**：
- DNS 记录已正确配置
- 邮箱服务已正常启用
- 已测试邮件转发功能
